# df命令

df-含义：

df - report file system disk space usage

报告文件系统磁盘空间使用情况

# 命令介绍

df			// 查看磁盘使用情况

Filesystem     1K-blocks    Used Available Use% Mounted on
/dev/vda1       57776288 1261228  54130472   3% /

Filesystem		// 文件系统（磁盘分区名称）
1K-blocks		// 磁盘总大小（单位K）
Used			// 已使用（单位K）
Available		// 未使用（单位K）
Use%			// 已使用百分比
Mounted on		// 挂载点

# 参数介绍

参数 意义
-l 合理化显示大小单位
-i 查看磁盘Inode使用情况
-m 已MB单位显示大小

小知识：

Byte(字节)

B是一个电脑存储的基本单位(字节),1个英文字符是1个字节,也就是1B，1个汉字为2个字符，也就是2B

1TB == 1024GB

1GB == 1024MB

1MB == 1024KB

1KB == 1024B
